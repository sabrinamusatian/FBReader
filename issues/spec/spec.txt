
Описание плагина/программы Bookshelf для FBReader.

Плагин представляет собой "окно" в области чтения, визуально отображающее содержимое библиотеки FBReader'а в различных режимах.

1. Базовый режим - отображение библиотеки в виде "плиток" (картинка + текст) в каком-то порядке.
2. Разделение библиотеки по жанрам(каким-либо другим представлениям) (возможно диалоговое окно, где пользователь выбирает жанр) - отображается часть библиотеки заданного жанра (? мета_данные книги ?).
3. Создание пользователем собственных разделов (? соответственно сериализация этих данных).


? Доступ к библиотеке ? 
а. Разобрать способ хранения книг в FBReader'e или
б. Кастомное решение с вызовом диалогового окна для чтения книги.


Режим полки - один из режимов ZLView, выбираемых из основного приложения. Переход в него из основного приложения осуществляется по кнопке на тулбаре FBReader'a. У режима есть свое диалоговое окно с настройками(включать в FBReader видимо сложнее), (а также возможно кнопка при нажатии которой выезжают возможности выбора режимов полки).
